## 十一、💰 成本优化技巧

> [!TIP]
> 1. **Batch调用半价** — 大部分千问系列支持，非实时场景必用
> 2. **上下文缓存** — 命中缓存享折扣（隐式20%/显式10%原价）
> 3. **阶梯计费** — 短输入更便宜，尽量控制单次请求Token数≤32K
> 4. **免费额度** — 每个模型开通后90天内有100万Token免费额度
> 5. **思考模式按需开** — 简单任务用非思考模式，输出价格显著更低
> 6. **DeepSeek-V3.2** — 第三方最具性价比（2元入/3元出，685B满血）
