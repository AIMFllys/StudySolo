# Wiki 内容目录

> 本目录由小陈负责填充。骨架由 Phase 5 第四波创建。
> 文档格式：Markdown（.md）
> 渲染路径：`/wiki/<path>` → 读取本目录对应文件

## 目录结构

```
wiki-content/
├── getting-started/
│   └── quick-start.md      # 快速开始
├── guides/
│   └── creating-workflows.md  # 创建工作流
└── README.md               # 本文件
```

## 新增文档说明

在对应子目录下新建 `.md` 文件即可，路由自动生成。
例如：`guides/advanced-nodes.md` → `/wiki/guides/advanced-nodes`
