# StudySolo 2026-03-24 产品更新履历

## 【面向用户】

### v0.5.0 对话意图识别网络与全新「手记美学」工作台架构
我们致力于将 StudySolo 打造成您的终极数字思维工坊。本次里程碑级别更新，我们将单调的左侧导航全面重塑为充满温度的“手绘笔记纸张”风格，同时赋予了 AI 助理全知全能的协同构建能力。它不再只局限于文字上的问答答疑，而是能直接帮您接管复杂幕后的隐形副手。

1. **全新视觉范式：全域「科研手札」美学深度重构 (Antigravity-Style UX)**：
   - 生产力工具不应是冰冷的铁盒。抛弃了沉重晦暗的边框与呆板的阴影结构，StudySolo 迎来了焕然一新的 `Paper & Ink` 质感升级。整个应用架构体系适配了极具高级感的纸质美学（`node-paper-bg` 规范），即使在夜间模式（Dark Mode）下也能带给您极为顺滑、护眼的沉浸阅读交互体验。
   - 整个对话交互区域正式引入专业的类科研 Markdown/LaTeX 图文混排排版支持体系，数学公式推演、高难度代码等均可完美剥壳呈现，专业质感尽显无遗。
2. **懂您所想的 AI：多维意图路由（Intent Routing）系统落地**：
   - 专属 AI 迎来了“理解力”跨越。过去，AI 助理只能给到文字反馈；如今得益于独家的“意图识别网络（Intent Routing）”，它可以秒级区分您的真实诉求：是单纯聊天（Chat）、全局规划（Plan），抑或是要直接绘制图纸（Build/Modify）。
   - **“说到做到”的画布交互**：您只需发出一道需求指令，工作流面板上的节点便能响应您的召唤自行动作。不管是全屏绘制，或是为您新增指定控制节点，AI 都能自主包办，释放您的双手。
3. **高阶环境集线中心：MCP 本地网关支持**：
   - 专为深度极客与进阶使用者，在设置与钱包界面内开拓了全新 MCP（Model Context Protocol）协议本地化集成配置区。您可以连接外部知识环境进行高维业务自定义延伸。

## 【面向开发者】

### v0.5.0 原生级 SSE 流式通信穿透与工程基建重构体系 (v2.0)
本轮系统更新在 Github 审查流中完成了极高质量的重构，包含 `feat: fix streaming pipeline ...` 等近 10 项主要提交记录，并清除了大量积弊，将前端侧边栏应用性能提升至更高标准。

1. **后端 ASGI 协议重构与原生流控 (SSE Proxy Pipeline Fix)**：
   - **剥离陈旧约束**：彻底废弃旧有依赖 FastAPI 常规响应与过度包装的阻塞型流式链路。
   - **原生流穿透升级**：后端改用 Pure ASGI 标准下发 Server-Sent Events 流，并在 Next.js 设立低依赖的 App Router SSE Proxy。这一改动彻底消弭了 AI 长文本由于网关超时导致的隐性中断及 `504` 灾难。前端同步利用原生增强的 `useStreamChat` 高容错转换逻辑实现了 100% 的字符拦截消费。
2. **意图路由调度分发架构设计 (AI Protocol v1.1)**：
   - **分轨处理设计**：开发了全新的中间件路桥。在组合提示词分发 LLM 之前，系统会预推断触发界域（`CHAT/PLAN/BUILD/MODIFY/ANALYZE`），然后依据意图池定向拼合特定动态系统提示词（`dynamic_system_prompts`）以及 Schema 限制，从而让单个输入域成功拓展为了图引擎命令解析器。
3. **前端视觉债务全量清洗 (14 Core Issues Resolved)**：
   - 执行严酷的代码体检 `UI 组件审查`，重剑斩落下游产生的 14 项 UI 深层隐患。彻底抹除了散落在不同业务面板（含钱包、协作侧边栏等）中的死代码调用。合并与精简出极简的 `dark:xxx` 模式支持向量，大幅降低页面重载运算强度。
4. **共享知识规范升级：`project-context v2.0` 正式定案释放**：
   - 我们在 `docs/shared` 目录下升级投递了深度底层系统解析架构文档 2.0 版本，专门收录了本次 AI 聊天系统的调度解构及模块职责分工全貌。这是未来整个项目架构延续扩展的唯一锚点事实与法典。
