# StudySolo 2026-03-18 产品更新履历

## 【面向用户】

### v0.4.0 画布交互底座重塑与全新科研美学视觉
我们很高兴为您推送这一重磅更新。本次迭代深度重塑了 StudySolo 的版式美学，上线了全套工作流交互工具箱，同时重新定义了节点之间的通讯拓扑规范，只为带给您更专业、专注的操作体验。

1. **视觉跃升：全局「科研美学」字体体系实装**：
   - 为了贴合严肃学习与深度思考的场景，我们对全站 UI 进行了深度排版调优。全新引入的科研级标准体系字体，兼顾了数字与中英文在极小字号下的锐利可读性（Legibility），搭配恰到好处的留白与手绘纸质纹理（Ink & Parchment），为您呈现典雅且舒适的阅读环境。
2. **全能画布：控制台底座与「右键扩展菜单」**：
   - 工作流编排从未如此称手。画布增配了高级控制底座菜单，原生支持缩放微调、一键全览适应（Fit-to-View）、焦点点击追踪及最小化隐藏等操作功能。
   - 上线全新节点上下文（Context Menu）面板，只需对节点或空白处轻击右键，即可呼出专属快捷指令浮层。
3. **节点通信机制革新与连线逻辑贯通**：
   - 修复并优化了节点间连接线在复杂层级下的缠绕与误触判定机制。全新的连线追踪引擎极大减少了“失焦”、“错连”问题，为您接下来的高阶树状分支编排提供强大基石支撑。
4. **全新侧边栏体验：AI 智能助手与钱包积分体系升级**：
   - 为了打破操作边界，AI 对话浮窗与个人资产钱包（包含积分明细、会员定价显示等）现已深度集成至侧边滑动抽屉中。您可以在调整工作流逻辑的同时，与 AI 进行连续对答，或者查阅实时账户状态与会员等级机制，无需再跳转离开当前沉浸视界。

## 【面向开发者】

### v0.4.0 画布基建、侧边栏架构设计与字体渲染工程
基于 Github 提交记录追踪，本次更新涵盖了超过 6 个合并周期的核心补丁代码（`feat:优化AI和钱包页面`、`feat:优化画布地图` 等），核心重构模块包括：

1. **统一拓扑图引擎与连接路由修复 (Phase 2)**：
   - 彻底翻新 React Flow 的底层拓扑节点通信系统。重新分配并标准化了 `sourceHandle` 与 `targetHandle` 的八向挂载锚点系统（Left、Right、Top、Bottom 双向进出），解决了部分连线场景下的渲染错位与 Z-Index 重叠异常（连接机制至 P2 化完成）。
   - 为后续（Phase 3）的 `logic_switch`（条件分支）及 `loop_group`（循环容器）高阶逻辑连线提供了标准 API 预埋。
2. **前端路由扩展与 Sidebar 模块化封装**：
   - 全面构建侧导航控制抽屉模块（Sidebar）。新增 `AIPanel` 聚合态视图及 `WalletPanel` 资管视图，采用基于 Zustand 的状态树拦截驱动开启控制，实现全局跨路由层即时挂载与卸载。并优化了用户头像组件映射会员等级（Tier）的数据绑定链路。
3. **全局 Font Rendering（字体渲染）优化**：
   - 整合并实装全新的全局字体库 `(font)`，重写了 `tailwind.config.ts` 中的 `fontFamily` 映射解析器。消解了以往不同组件使用杂乱 Web-Safe 字体的碎片化问题，统一使用科研出版级的严谨行高倍率与字体基线。
4. **画布外围交互集成 (Control Board & Menus)**：
   - 开发并补齐了 `React Flow Panel` 的拓展能力。集成 `ContextMenu` 组件进行绝对定位锚定（Absolute Pos Anchor），为 Node 和 Canvas 提供独立的快捷支持控制层面板，收口原本悬浮杂乱的画布内选项按钮。
