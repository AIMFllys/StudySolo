#### v0.1.009 📏 5 份核心工程规范文档创建 (integration-fixes Phase 0)
为提高多人协作和辅助工具的代码一致性，创建了 5 份可机器读取的工程规范文档：
- **`docs/architecture.md`**（项目地图）：技术栈选型表（前端 Next.js 16.1 / 后端 FastAPI / 数据库 Supabase）、模块划分与职责边界（8 个前端模块 + 5 个后端模块，明确各模块"管什么、不管什么"）、数据流向图（用户请求 → Next.js → FastAPI → AI 平台 / Supabase 的完整链路）、目录结构到二级、数据库表清单、端口约定 2037/2038。
- **`docs/naming.md`**（命名规范）：前端 TypeScript 命名模式（布尔值 `is/has` 前缀、事件 `handle` 前缀、回调 `on` 前缀）、后端 Python 命名（snake_case 变量/函数、PascalCase 类）、文件命名规则（组件 PascalCase、Hook kebab-case、后端 snake_case）、API 字段前后端映射（后端 `user_id` → 前端 `userId`）、10 条禁止写法反例清单、项目专有术语统一表。
- **`docs/design.md`**（UI 设计规范）：精确色彩 Token（primary `#6366F1`、accent `#10B981`、background-dark `#020617` 等十六进制值，非含糊的"蓝色系"）、间距系统 spacing-1 到 spacing-8、圆角系统 radius-sm 到 radius-full、毛玻璃效果三级定义（glass-panel / glass-card / glass-active 各自的 `backdrop-blur` 和透明度参数）、阴影系统（shadow-glass / shadow-glow）、字体规格 6 个档位（Inter + Noto Sans SC）、Shadcn/UI + Material Symbols 使用约定、响应式 4 个断点定义、各页面和节点的设计规范。
- **`docs/api.md`**（接口契约）：请求/响应统一格式（成功/错误/分页结构的 JSON 模板）、Cookie-based JWT 鉴权流程详解、已有 12 个端点列表（认证 5 个 + 工作流 6 个 + AI 生成 1 个）、核心数据结构定义（UserInfo、WorkflowMeta、WorkflowContent、4 种 SSE 事件类型）、RESTful 命名规则、安全约定（RLS、限流、注入防护）。
- **`docs/progress.md`**（当前状态文档）：已完成模块清单、当前任务状态表、6 项已知技术债、下一步计划。每次新开开发会话时作为首要上下文参考。

#### v0.1.010 🎨 整站暗色 Glass Morphism 主题重设计 (integration-fixes Phase 1)
根据 `docs/技术指导/StudySolo概念图/` 中的设计图，对整站 UI 进行重设计：
- **全局色彩系统重写**（`globals.css`）：将原有 OKLCH 灰度色彩变量整体替换为暗色主题 Token（background-dark `#020617`、surface-dark `#0F172A`、primary `#6366F1`）。新增 `glass-panel`（`backdrop-blur-md bg-white/5`）、`glass-card`（`backdrop-blur bg-white/[0.03]`）、`glass-active`（`bg-primary/10 border-primary/30`）三个全局 CSS 类。新增 `bg-grid-pattern` 网格点阵背景、`shadow-glow` 紫色发光阴影、自定义暗色滚动条样式。清理无关的 Geist/Vercel 设计系统残留变量。
- **Sidebar**：背景 `#020617`，工作流列表项 hover `bg-white/5`，活跃项 glass-active，运行中工作流绿色脉冲指示灯（`animate-pulse`），宽度 lg 280px / md 64px 图标模式 / md 以下隐藏。
- **Navbar**：glass-panel 背景 h-14，Logo 改为 primary 色 bolt 图标 + 渐变文字，搜索框 `bg-slate-900/50` 圆角胶囊形，新建按钮 primary 色 + shadow-glow，头像 hover 时 primary 光环。
- **MobileNav**：底部固定 glass-panel 导航栏，活跃项 primary 高亮，仅 md 以下显示。
- **登录/注册页**：改为左右分栏——左侧品牌展示区（`#050B1D` 背景 + 星空背景动画 + 代码雨特效 + 品牌标语），右侧 glass 风格输入框（`bg-surface-dark/50`）+ GitHub/Google 社交登录按钮。
- **WorkflowCanvas 画布**：背景 `#050B1D` + grid-pattern 点阵，节点卡片 glass-card / glass-active，连线 primary 色渐变 + 脉冲动画，浮动工具栏 `bg-slate-900/80 backdrop-blur-md`，缩放控件底部右侧圆角胶囊。
- **Landing 首页**：Deep Midnight Navy 背景 + 装饰性光晕，CTA 按钮 primary 色 + shadow-glow。
- 逐页对比概念图进行视觉一致性验证，确认 sm/md/lg/xl 四个响应式断点下的布局表现。
